Description describeMismatch(item, Description mismatchDescription, Map matchState, bool verbose) #

inherited from BaseMatcher

Generates a description of the matcher failed for a particular item, by appending the description to mismatchDescription. It does not check whether the item fails the match, as it is only called after a failed match. There may be additional info about the mismatch in matchState. The base matcher does not add anything as the actual value is typically sufficient, but matchers that can add valuable info should override this.
